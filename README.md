# Pokémon App

App feito em React Native (Expo) onde o usuário se cadastra, faz login, vê uma lista de Pokémons e monta seu time (máximo 5).

## Como abrir o projeto

bash
npm install
npx expo start


## Telas

- *Login / Cadastro* — criar conta e entrar
- *Dashboard* — galeria com todos os Pokémons
- *Meu Time* — escolher até 5 Pokémons pro time, com opção de trocar
- *Perfil* — mostra vitórias e derrotas do jogador

## Observação

A senha do cadastro precisa ter letra maiúscula, número e símbolo (ex: Senha@123).
