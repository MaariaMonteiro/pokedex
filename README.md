# Pokémon App

Aplicativo desenvolvido em React Native (Expo) onde o usuário pode criar uma conta, fazer login, visualizar uma lista de Pokémons e montar um time com até 5 integrantes.

## Funcionalidades

- Cadastro e login de usuários
- Visualização da lista de Pokémons
- Criação e gerenciamento de um time com até 5 Pokémons
- Substituição de Pokémons do time
- Perfil do jogador com informações de vitórias e derrotas

## Como executar o projeto

1. Clone o repositório:
   ```bash
   git clone <URL_DO_REPOSITORIO>
   ```

2. Instale as dependências:
   ```bash
   npm install
   ```

3. Inicie a aplicação:
   ```bash
   npx expo start
   ```

## Telas

- **Login / Cadastro:** criação de conta e autenticação.
- **Dashboard:** exibição da lista de Pokémons.
- **Meu Time:** seleção e gerenciamento de até 5 Pokémons.
- **Perfil:** visualização das estatísticas do jogador, incluindo vitórias e derrotas.

## Tecnologias utilizadas

- React Native
- Expo
- TypeScript
- React Navigation
- API Pokémon
